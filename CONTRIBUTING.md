# Contributing to PHX

Thanks for your interest in PHX. PHX is currently a **technical preview**, and
this note explains what we can and cannot accept at this stage.

## Bug reports and feedback — welcome

Please use the issue tracker to report bugs, suggest features, or share how you
use PHX. Reproducible examples are especially appreciated. Feedback of this kind
does not require any agreement.

## Code contributions — not accepted during the technical preview

We are **not accepting external code contributions** (pull requests / patches)
during the technical preview. Please do not submit code changes yet.

The reason is licensing. PHX is released under the PHX Preview License (see
[`LICENSE.txt`](LICENSE.txt)), a source-available license, and HUMUSOFT also
intends to offer a separate commercial license. To keep the option of including a contribution in
the commercially licensed edition, HUMUSOFT must hold the necessary rights to
that contribution. Before we accept any external code, we will therefore
introduce a **Contributor License Agreement (CLA)**. A sign-off such as a DCO is
not sufficient for this purpose.

When external contributions open up, this document will be updated with the CLA
and the process.
