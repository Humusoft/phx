phx.base.Shape

Shape superclass

Abstract class  phx.base.Shape

Description

phx.base.Shape is the abstract superclass for all shapes that define the visual and collision geometry of a phx.Body. Concrete shapes in the phx.shape.* package derive from this class and implement how they are drawn and how their mass and moments of inertia are computed. The class also provides an optional skeletal representation through the Skelet* properties.

You normally do not instantiate this class directly; you assign a concrete shape to a body's Shape property, either as an object or as a cell array.

Concrete Shapes

The phx.shape.* package provides the ready-to-use geometry a body can wear, including:

Box, Sphere, Cylinder, Cone, Capsule, Globe, Mesh, Extrusion, Revolution, Rock, Terrain, OBJ and STL.

Examples

Assign a shape to a body

clf; view(3); axis equal; grid on; camlight headlight;

% As a cell array: shape name followed by its name-value pairs
b1 = phx.Body("Shape", {"Box", "Size", [2 1 1], "Color", [0.4 0.6 1]});

% As a shape object
b2 = phx.Body("Shape", phx.shape.Sphere("Diameter", 0.5, "Density", 50));

Add a skeleton to a shape

A = phx.Body("Type", "static", ...
    "Shape", {"Box", "SkeletPoints", [1.5 0 0], "SkeletStyle", "rays"});
