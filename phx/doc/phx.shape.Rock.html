phx.shape.Rock

Rock shape

Class  phx.shape.Rock  |  Superclasses: phx.base.Shape, phx.base.ShapeMesh

Description

phx.shape.Rock is a procedurally generated irregular rock created by perturbing an icosahedron. Roundness controls how close the result is to an ideal sphere and Diameter (or Radius) sets its size. The collision shape is the convex hull of the generated mesh, while the mass properties are modeled as an ideal sphere of the given Density.

Creation

Syntax

shape = phx.shape.Rock shape = phx.shape.Rock(Name,Value)

Description

shape = phx.shape.Rock creates a unit rock with default parameters. phx.shape.Rock(Name,Value) sets properties with name-value arguments.

Properties

Geometry

PropertyType / DefaultDescription
Diameter1Rock diameter, m.
Radiusdouble (dependent)Rock radius, m. Setting it updates Diameter.
Roundness0.75Roundness factor relative to the ideal sphere (0–1, higher is rounder).
Density1000Volumetric density, kg/m³.

Appearance properties are inherited from phx.base.ShapeMesh and skeletal properties from phx.base.Shape. Rock uses the "flat" style by default.

Examples

A scattering of boulders

for k = 1:5
    phx.Body("Position", [k 0 3], ...
        "Shape", {"Rock", "Diameter", 0.8, "Roundness", 0.6, "Color", [0.5 0.45 0.4]});
end

sim = phx.Simulation;
sim.step(3, 300, 1);
