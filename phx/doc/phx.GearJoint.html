phx.GearJoint

Gear joint

Class  phx.GearJoint  |  Superclass: phx.base.Joint

Description

phx.GearJoint couples the rotations of two bodies so that their angular velocities keep a defined ratio — the constraint of a pair of meshing gears. The gear axes are defined in the local space of each body.

Creation

Syntax

joint = phx.GearJoint(bodyA,bodyB) joint = phx.GearJoint(___,Name,Value)

Description

joint = phx.GearJoint(bodyA,bodyB) couples two bodies with their rotation axes aligned to the local Z axis of each body and a unit gear ratio. Set the axes with AxisA/AxisB and the ratio with Ratio.

Input Arguments

ArgumentTypeDescription
bodyA, bodyBphx.Body (scalar)The two coupled bodies.

Properties

PropertyType / DefaultDescription
AxisA[0 0 1]Rotation-axis direction in the local space of body A.
AxisB[0 0 1]Rotation-axis direction in the local space of body B.
Ratio1Gear ratio between the angular velocities of the two bodies.
OverlayfalseDraw the joint as an overlay (always on top).

Reaction feedback and the MutualCollisions switch (collisions between the connected bodies, default false) are inherited from phx.base.Joint.

Examples

Couple two wheels with a 2:1 ratio

clf; view(3); axis equal; grid on; camlight headlight;

w1 = phx.Body("Position", [0 0 0], "Shape", {"Cylinder", "Radius", 1, "Height", 0.3});
w2 = phx.Body("Position", [1.5 0 0], "Shape", {"Cylinder", "Radius", 0.5, "Height", 0.3});

phx.RevoluteJoint(phx.Body("Type", "static"), w1);
phx.GearJoint(w1, w2, "Ratio", 2);

sim = phx.Simulation(gca, "Gravity", [0 0 0]);   % no gravity - the free wheel keeps its place
w1.AngularVelocity = [0 0 5];
sim.step(4, 400, 1);
delete(sim);
