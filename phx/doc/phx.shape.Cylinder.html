phx.shape.Cylinder

Cylinder shape

Class  phx.shape.Cylinder  |  Superclasses: phx.base.Shape, phx.base.ShapeMesh

Description

phx.shape.Cylinder is a cylinder geometry. The visual appearance is a revolved profile with an adjustable number of segments; the collision shape and mass properties are modeled as an ideal cylinder independent of the visual detail.

Creation

Syntax

shape = phx.shape.Cylinder shape = phx.shape.Cylinder(Name,Value)

Description

shape = phx.shape.Cylinder creates a unit cylinder with default parameters. phx.shape.Cylinder(Name,Value) sets properties with name-value arguments.

Properties

Geometry

PropertyType / DefaultDescription
Diameter1Cylinder diameter, m.
Radiusdouble (dependent)Cylinder radius, m. Setting it updates Diameter.
Height1Cylinder height, m.
Axis"z"Modeling axis: "x", "y" or "z".
Segments24Number of segments around the circumference.
Density1000Volumetric density, kg/m³.

Appearance properties are inherited from phx.base.ShapeMesh and skeletal properties from phx.base.Shape.

Examples

Simple falling body

phx.extra.Viewer("clear", "DefaultCameraPosition", [10 10 10]);

w = phx.Body("Position", [0 0 1], ...
    "Shape", {"Cylinder", "Radius", 1, "Height", 0.3, "Axis", "x"});

sim = phx.Simulation;
sim.step(3, 300, 1);
