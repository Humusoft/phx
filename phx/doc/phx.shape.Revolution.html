phx.shape.Revolution

Revolution shape

Class  phx.shape.Revolution  |  Superclasses: phx.base.Shape, phx.base.ShapeMesh

Description

phx.shape.Revolution revolves a user-defined 2-D Profile about the modeling axis to form a solid of revolution. The collision shape is a convex or concave envelope (Envelope); the mass properties are computed from the actual triangular mesh.

Creation

Syntax

shape = phx.shape.Revolution shape = phx.shape.Revolution(Name,Value)

Description

shape = phx.shape.Revolution(Name,Value) creates a solid of revolution from the given Profile.

Properties

Geometry

PropertyType / DefaultDescription
ProfileM×2 doubleProfile revolved about the modeling axis; each row is a 2-D point (axial, radial).
Axis"z"Modeling axis: "x", "y" or "z".
Segments45Number of segments around the revolution.
Envelope"convex"Collision envelope: "convex" or "concave".
Density1000Volumetric density, kg/m³.

Appearance properties are inherited from phx.base.ShapeMesh and skeletal properties from phx.base.Shape.

Examples

Revolve a profile into a vase

phx.extra.Viewer("clear", "DefaultCameraPosition", [10 10 10]);

t = linspace(0, 1, 20)';
profile = [t, 0.3 + 0.2*sin(t*pi)];     % [axial, radial]

b = phx.Body("Shape", phx.shape.Revolution("Profile", profile, "Envelope", "concave"));

sim = phx.Simulation;
sim.step(2, 200, 1);
See also phx.Body, phx.shape.Extrusion, phx.shape.Cylinder, phx.base.Shape
