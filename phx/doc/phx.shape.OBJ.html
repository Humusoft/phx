phx.shape.OBJ

OBJ imported shape

Class  phx.shape.OBJ  |  Superclasses: phx.base.Shape, phx.base.ShapeMesh

Description

phx.shape.OBJ loads its visual appearance from a Wavefront OBJ file specified by the Source property, including its texture coordinates. The mesh can be scaled (Scale) and recentred to its bounding-box origin (Centered). The collision shape is selected by Envelope and can be a bounding box, cylinder, sphere, convex hull or concave triangle mesh; mass properties are modeled from the bounding box and Density.

Creation

Syntax

shape = phx.shape.OBJ shape = phx.shape.OBJ(Name,Value)

Description

shape = phx.shape.OBJ("Source",file) loads a mesh from an OBJ file. Set Envelope to choose how the collision shape is approximated.

Properties

Geometry

PropertyType / DefaultDescription
SourcestringOBJ source file name. Setting it loads the mesh.
Scale[1 1 1]Scale factor applied to the imported mesh.
CenteredfalseMove the origin to the centre of the bounding box.
Envelope"box"Collision envelope: "box", "cylinder", "sphere", "convex" or "concave".
Density1000Volumetric density, kg/m³ (applied to the bounding box).

Appearance properties are inherited from phx.base.ShapeMesh and skeletal properties from phx.base.Shape.

Examples

Import a textured model with a convex collision hull

shape = phx.shape.OBJ("Source", "teapot.obj", "Scale", [1 1 1]*0.5, ...
    "Centered", true, "Envelope", "convex");

b = phx.Body("Position", [0 0 3], "Shape", shape);

sim = phx.Simulation;
sim.step(3, 300, 1);
