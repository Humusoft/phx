phx.FixedJoint

Fixed joint

Class  phx.FixedJoint  |  Superclass: phx.base.Joint

Description

phx.FixedJoint realizes a kinematic constraint with zero degrees of freedom — a rigid connection that locks the relative pose of two bodies at the given connection points.

Creation

Syntax

joint = phx.FixedJoint(bodyA,bodyB) joint = phx.FixedJoint(___,Name,Value)

Description

joint = phx.FixedJoint(bodyA,bodyB) rigidly connects two bodies at their origins. Set custom connection points with PointA and PointB, or the full joint frames with TransformA and TransformB.

Properties

PropertyType / DefaultDescription
TransformAeye(4)Homogeneous transformation of the joint frame relative to body A.
TransformBeye(4)Homogeneous transformation of the joint frame relative to body B.
PointA[0 0 0]Connection point in the local space of body A; the translation part of TransformA (dependent).
PointB[0 0 0]Connection point in the local space of body B; the translation part of TransformB (dependent).
EulerAnglesA[0 0 0]Euler angles (z→y→x order) of the joint frame relative to body A; the rotation part of TransformA (dependent).
EulerAnglesB[0 0 0]Euler angles (z→y→x order) of the joint frame relative to body B; the rotation part of TransformB (dependent).

Reaction feedback is inherited from phx.base.Joint.

Examples

Rigidly weld two bodies

a = phx.Body("Position", [0 0 2]);
b = phx.Body("Position", [1 0 2]);

phx.FixedJoint(a, b, "PointA", [0.5 0 0], "PointB", [-0.5 0 0]);

sim = phx.Simulation;
sim.step(2, 200, 1);
