phx.Body

Physical body

Class  phx.Body  |  Superclass: phx.base.Object

Description

phx.Body is the principal object of PHX. It represents every movable object in the virtual world: it carries the mass and inertia, holds the kinematic state (pose and velocities), wears a geometric Shape for visualization and collision, and is the body that joints, springs and force elements attach to.

A body can be dynamic (driven by forces and collisions), static (immovable, e.g. ground) or kinematic (moved by script, unaffected by forces). Many kinematic properties are dependent and read from or write to the engine directly while the simulation is built.

Creation

Syntax

body = phx.Body body = phx.Body(ax) body = phx.Body(___,Name,Value)

Description

body = phx.Body(ax) creates a physical body and draws its visual representation into the axes ax (default gca). Pass [] when no visualization is needed; the target axes can also be set later through the ParentAxes property.

A body with no explicit shape is created as a unit Box.

body = phx.Body(___,Name,Value) sets properties using name-value arguments, for example phx.Body("Type","static","Position",[1 0 0]).

Input Arguments

ArgumentTypeDescription
axaxes | UIAxes | [] Target axes for the body graphics. Use [] for a headless body.

applyForce

applyForce(body,force) applyForce(body,force,point) applyForce(body,force,point,isLocalForce,isLocalPoint)

Applies force (N) acting during the next simulation step only. Without point the force acts at the body origin; otherwise it acts at point (local space by default). The two logical flags select local (true, default) or world space for the force and the point respectively.

applyTorque

applyTorque(body,torque) applyTorque(body,torque,isLocalTorque)

Applies torque (N·m) during the next step. isLocalTorque selects local (default) or world space.

offset

M = offset(body,translation) M = offset(body,translation,rotation)

Returns a transform with translation and rotation relative to the body itself. rotation is interpreted by size: scalar → angle about Z, 1×3 → Euler angles, 1×4 → axis-angle.

overlay

overlay(body,type) overlay(body,type,state)

Toggles (or sets "on"/"off") an auxiliary overlay. type is one of "center", "system", "bbox", "edit" or "hull".

Examples

Create a dynamic body and a ground plane

clf; view(3); axis equal; grid on; camlight headlight;

ground = phx.Body("Type", "static");
box    = phx.Body("Position", [0 0 3], ...
                 "Shape", {"Box", "Size", [1 1 1], "Color", [0.4 0.6 1]});

sim = phx.Simulation;
sim.step(2, 200, 1);

Drive a body with a transient force

box.applyForce([0 0 50]);              % upward kick, world frame
box.applyForce([10 0 0], [0 0 0.5]);   % off-centre push, local point
sim.step(0.1, 20, 1);

Read state during the run

v = box.LinearVelocity;
E = box.Energy;                        % kinetic energy (J)
bb = box.boundingBox;                  % [min; max] world corners
Tip. Use storeState before a run and restoreState afterwards to replay a scenario from identical initial conditions without rebuilding it.
