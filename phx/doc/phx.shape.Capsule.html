phx.shape.Capsule

Capsule shape

Class  phx.shape.Capsule  |  Superclasses: phx.base.Shape, phx.base.ShapeMesh

Description

phx.shape.Capsule is a capsule geometry — a cylinder capped by two hemispheres. The visual appearance is a revolved profile with an adjustable number of segments; the collision shape and mass properties are modeled as an ideal capsule independent of the visual detail. The Height is the length of the cylindrical part between the two hemispherical caps.

Creation

Syntax

shape = phx.shape.Capsule shape = phx.shape.Capsule(Name,Value)

Description

shape = phx.shape.Capsule creates a unit capsule with default parameters. phx.shape.Capsule(Name,Value) sets properties with name-value arguments.

Properties

Geometry

PropertyType / DefaultDescription
Diameter1Capsule diameter, m.
Radiusdouble (dependent)Capsule radius, m. Setting it updates Diameter.
Height1Length of the cylindrical part (between caps), m.
Axis"z"Modeling axis: "x", "y" or "z".
Segments24Number of segments around the circumference.
Density1000Volumetric density, kg/m³.

Appearance properties are inherited from phx.base.ShapeMesh and skeletal properties from phx.base.Shape.

Examples

Simple falling body

phx.extra.Viewer("clear", "DefaultCameraPosition", [10 10 10]);

c = phx.Body("Position", [0 0 2], ...
    "Shape", {"Capsule", "Radius", 0.25, "Height", 1, "Axis", "x"});

sim = phx.Simulation;
sim.step(3, 300, 1);
