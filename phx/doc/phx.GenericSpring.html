phx.GenericSpring under development

Generic spring

Class  phx.GenericSpring  |  Superclass: phx.base.Joint

Under development. This class is functional but still evolving; its API and behaviour may change in a future release.

Description

phx.GenericSpring realizes a six-degree-of-freedom compliant constraint — a spring–damper acting independently on each linear and angular axis between two bodies. Unlike phx.Spring, which acts along a single line, the generic spring constrains the full relative pose with per-axis stiffness and damping.

Creation

Syntax

spring = phx.GenericSpring(bodyA,bodyB) spring = phx.GenericSpring(___,Name,Value)

Description

spring = phx.GenericSpring(bodyA,bodyB) creates a compliant 6-DOF connection between two bodies at their origins. Set custom connection points with PointA and PointB.

Properties

PropertyType / DefaultDescription
PointA[0 0 0]Connection point in the local space of body A.
PointB[0 0 0]Connection point in the local space of body B.
LinearLimits[0 0 0; 0 0 0]Lower and upper linear limits per axis (2×3).
AngularLimits[0 0 0; 0 0 0]Lower and upper angular limits per axis (2×3), rad.

Examples

Compliant mount between two bodies

a = phx.Body("Type", "static", "Position", [0 0 2]);
b = phx.Body("Position", [0 0 2]);

phx.GenericSpring(a, b);

sim = phx.Simulation;
b.applyForce([0 0 -30]);
sim.step(3, 600, 5);
