phx.shape.Box

Box shape

Class  phx.shape.Box  |  Superclasses: phx.base.Shape, phx.base.ShapeMesh

Description

phx.shape.Box is the geometry of a rectangular block worn by a phx.Body. The visual appearance is a simple triangular mesh; the collision shape and mass properties are modeled as an ideal block.

Creation

Syntax

shape = phx.shape.Box shape = phx.shape.Box(Name,Value)

Description

shape = phx.shape.Box creates a unit box with default parameters. phx.shape.Box(Name,Value) sets properties with name-value arguments.

A shape is normally passed to a body's Shape property, either as an object or as a cell array {"Box",Name,Value,…}.

Properties

Geometry

PropertyType / DefaultDescription
Size[1 1 1]Box size [X Y Z], m.
Density1000Volumetric density, kg/m³ (used to compute mass and inertia).

In addition, every shape inherits appearance properties (Color, Material, Style, Texture, …) from phx.base.ShapeMesh and skeletal properties (SkeletPoints, SkeletStyle) from phx.base.Shape. Box uses the "flat" style by default.

Examples

A colored block as a body

b = phx.Body("Position", [0 0 3], ...
    "Shape", {"Box", "Size", [2 1 0.5], "Color", [0.4 0.6 1]});

sim = phx.Simulation;
sim.step(2, 200, 1);
