phx.Thruster

Thrust actuator

Class  phx.Thruster  |  Superclass: phx.base.Object

Description

phx.Thruster applies a body-fixed thrust force to a body. The mounting point and the thrust direction are defined in the local space of the body, so the thrust follows the body orientation — like an attached rocket engine, a propeller or a fan.

The actual thrust is MaxThrust·Throttle. Optionally it follows the throttle with a first-order lag (TimeConstant) and applies a reaction torque about the thrust axis (ReactionFactor) — the counter-torque of a spinning propeller. Throttle can be changed while the simulation runs, driven by a phx.Script or set from Simulink; the resulting Thrust can be recorded with a phx.Logger.

Creation

Syntax

thr = phx.Thruster(body) thr = phx.Thruster(___,Name,Value)

Description

thr = phx.Thruster(body) creates a thruster acting at the origin of the body and thrusting along its local Z axis. Adjust the mounting point, direction and thrust through the properties below.

Input Arguments

ArgumentTypeDescription
bodyphx.Body (scalar)The body the thruster is attached to.

Properties

PropertyType / DefaultDescription
Point[0 0 0]Mounting point in the local space of the body.
Direction[0 0 1]Thrust direction in the local space of the body (normalized to a unit vector).
MaxThrust1Maximal thrust force at full throttle, N.
Throttle0Throttle setting in [-1, 1] (negative reverses thrust).
TimeConstant0Time constant of the first-order throttle response, s (0 = immediate).
ReactionFactor0Reaction torque about the thrust axis per unit thrust, N·m/N.
ForceVectorSize0.001Exhaust-vector multiplication factor for drawing.
OverlayfalseDraw the exhaust vector as an overlay (always on top).
Thrustdouble (read-only)Current thrust force magnitude after the throttle lag, N.

Examples

Full-throttle thruster in zero gravity

clf; ax = gca; view(3); axis equal; grid on;
b = phx.Body(ax, "Shape", {"Sphere", "Diameter", 0.5, "Density", 50});

phx.Thruster(b, "MaxThrust", 10, "Throttle", 1);

sim = phx.Simulation(ax, "Gravity", [0 0 0]);
sim.step(4, 800, -1);
disp(b.LinearVelocity);

Throttle driven by a script, thrust logged

th = phx.Thruster(b, "MaxThrust", 10, "TimeConstant", 0.5, "ReactionFactor", 0.05);
phx.Script(th, {'Throttle', [0 1], [0 1], 'linear', 'nearest'});
lg = phx.Logger(th, "Frequency", 100, "Parameters", "Thrust");
