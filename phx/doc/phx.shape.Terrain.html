phx.shape.Terrain

Terrain heightfield

Class  phx.shape.Terrain  |  Superclasses: phx.base.Shape, phx.base.ShapeMesh

Description

phx.shape.Terrain is a surface generated from the Height matrix sampled over a rectangle of the given Size, optionally textured. The collision shape is a heightfield built in the engine from the same data. A terrain is typically used as a static body.

Creation

Syntax

shape = phx.shape.Terrain shape = phx.shape.Terrain(Name,Value)

Description

shape = phx.shape.Terrain(Name,Value) creates a terrain from a Height matrix sampled over a Size rectangle.

Properties

Geometry

PropertyType / DefaultDescription
Size[10 10]Length and width of the terrain rectangle, m.
Heightzeros(10)Height matrix sampled over the rectangle, m.

Appearance properties are inherited from phx.base.ShapeMesh and skeletal properties from phx.base.Shape.

Tip. Use a terrain as a "static" body so it stays fixed while dynamic bodies collide with it.

Examples

A rolling-hills ground

phx.extra.Viewer("clear", "DefaultCameraPosition", [10 10 10]);

[X, Y] = meshgrid(linspace(-5, 5, 40));
H = 0.6*sin(X).*cos(Y);

ground = phx.Body("Type", "static", ...
    "Shape", phx.shape.Terrain("Size", [10 10], "Height", H));
ball   = phx.Body("Position", [0 0 4], "Shape", {"Sphere", "Diameter", 0.5});

sim = phx.Simulation;
sim.step(4, 400, 1);
See also phx.Body, phx.shape.Box, phx.base.Shape
