phx.assembly.arena

Static arena — a floor plate enclosed by four walls

Function  phx.assembly.arena  |  Package: phx.assembly

Description

phx.assembly.arena builds a static floor with four walls around it, ready to keep dynamic bodies from escaping the scene. The arena origin lies at the middle of the floor surface, so by default the inner space spans −x/2..x/2 and −y/2..y/2 in the ground plane and 0..z upwards around the world origin. The parts are ordinary static bodies, so they can be restyled or retextured afterwards through the returned struct.

Creation

Syntax

parts = phx.assembly.arena() parts = phx.assembly.arena(Name,Value) parts = phx.assembly.arena(ax,___)

Description

parts = phx.assembly.arena() creates the arena in the current axes and returns a struct with the fields floor (the floor body) and walls (1×4 phx.Body, in the order −x, +x, −y, +y).

parts = phx.assembly.arena(ax,___) draws into the given axes instead of the current axes; an empty target ([]) creates the parts without graphics for headless simulations.

Options

OptionDefaultDescription
Size[2 2 0.5]Inner dimensions [x y z] of the arena — the usable floor area x-by-y and the inner height z of the walls.
Thickness0.1Thickness of the floor and wall plates.
Color[1 1 1]Common color of all parts.
Friction[0.5 0 0]Friction coefficients of all parts.
Position[0 0 0]World position of the arena origin (the middle of the floor surface).
Orientationeye(3)World rotation of the arena as a 3×3 rotation matrix. Same convention as phx.Body.Orientation.
EulerAngles[0 0 0]World rotation of the arena as Euler angles (z→y→x order), an alternative to Orientation.

Examples

Keep a bouncing ball in a box

clf; view(3); axis equal; grid on; camlight headlight;

parts = phx.assembly.arena("Size", [4 3 0.3]);
set(parts.walls, "Color", [0.4 0.55 0.7]);

phx.Body("Position", [0 0 1], "LinearVelocity", [3 2 0], ...
    "Shape", {"Sphere", "Diameter", 0.3});

sim = phx.Simulation;
sim.step(5, 1000, 5);
delete(sim);
