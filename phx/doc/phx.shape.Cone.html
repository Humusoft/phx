phx.shape.Cone

Cone shape

Class  phx.shape.Cone  |  Superclasses: phx.base.Shape, phx.base.ShapeMesh

Description

phx.shape.Cone is a cone geometry. The visual appearance is a revolved triangular profile with an adjustable number of segments; the collision shape and mass properties are modeled as an ideal cone independent of the visual detail.

Creation

Syntax

shape = phx.shape.Cone shape = phx.shape.Cone(Name,Value)

Description

shape = phx.shape.Cone creates a unit cone with default parameters. phx.shape.Cone(Name,Value) sets properties with name-value arguments.

Properties

Geometry

PropertyType / DefaultDescription
Diameter1Base diameter, m.
Radiusdouble (dependent)Base radius, m. Setting it updates Diameter.
Height1Cone height, m.
Axis"z"Modeling axis: "x", "y" or "z".
Segments24Number of segments around the base.
Density1000Volumetric density, kg/m³.

Appearance properties are inherited from phx.base.ShapeMesh and skeletal properties from phx.base.Shape.

Examples

Simple falling cone

phx.extra.Viewer("clear", "DefaultCameraPosition", [10 10 10]);

c = phx.Body("Position", [0 0 2], ...
    "Shape", {"Cone", "Diameter", 1, "Height", 1.5});

sim = phx.Simulation;
sim.step(3, 300, 1);
See also phx.Body, phx.shape.Cylinder, phx.base.Shape
