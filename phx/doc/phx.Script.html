phx.Script

Automation script

Class  phx.Script  |  Superclass: phx.base.Object

Description

phx.Script controls the values of selected parameters of other objects during the simulation. The value profile for each parameter is defined either as an interpolated curve (a vector of times and a matching array of values) or as a time-dependent expression given as a string. The profiles are evaluated before every simulation substep and written to the given property of every parent object — declarative, feedforward control ideal for kinematic bodies such as conveyors, arms or drive cams. For feedback control that reacts to the scene state, use phx.Function instead.

Creation

Syntax

script = phx.Script(parents,curve) script = phx.Script(parents,curve1,...,curveN)

Description

script = phx.Script(parents,curve) creates a script driving one parameter of all parent objects. Any number of curves can follow, each driving one parameter. Each curve is a cell array in one of two forms:

{"ParamName", timeVector, values} — an interpolated curve. Optionally append an interpolation method and an extrapolation method ({"ParamName", time, values, "pchip", "repeat"}); both default to "linear". The special extrapolation method "repeat" loops the curve periodically.

{"ParamName", "expression of t"} — a function of the simulation time t, e.g. {"EulerAngles", "[2*pi*t 0 0]"}.

Input Arguments

ArgumentTypeDescription
parentsphx object array or cell arrayThe objects whose parameters are driven. A cell array may mix objects of different classes.
curve1, ..., curveNcell (repeating)One profile per parameter, in either of the two forms above.

Properties

PropertyType / DefaultDescription
CurvesN×5 cellInterpolation curves as {'ParamName', timeVector, valuesVector, 'interpolation_method', 'extrapolation_method'} rows (filled from the constructor arguments).

Examples

Rotate a screw steadily by an expression of time

phx.Script(screw, {"EulerAngles", "[2*pi*t*0.5 0 0]"});

Prescribe the path of an arm by interpolated points

time     = [0 1 2 3];
position = [0 0 1; 1 0 1; 1 1 1; 0 1 1];
angle    = [0 0 0; 0 0 pi/2; 0 0 pi; 0 0 3*pi/2];

phx.Script(arm, {"Position", time, position, "pchip"}, ...
                {"EulerAngles", time, angle});

sim = phx.Simulation;
sim.step(3, 600, 2);
