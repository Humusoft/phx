phx.Monopole

Monopole interaction

Class  phx.Monopole  |  Superclass: phx.base.Object

Description

phx.Monopole assigns a monopole charge to selected bodies. All bodies in the group then interact pairwise through an inverse-square reaction force:

          c1 * c2
F = A * -----------
            r^2

where c1, c2 are the charges, r the distance, and A the attractivity sign. The interaction can be visualized as a vector field on a configurable grid.

Creation

Syntax

m = phx.Monopole(bodies) m = phx.Monopole(___,Name,Value)

Description

m = phx.Monopole(bodies) creates a monopole element and assigns it to all given bodies, forming a mutually interacting group. Provide one charge per body through the Charge property.

Input Arguments

ArgumentTypeDescription
bodiesphx.Body (1×n)Bodies forming the interacting group.

Properties

PropertyType / DefaultDescription
Chargen×1 doubleMonopole charge of each body (one per body).
Attractivity-1Polarity: -1 = opposite charges attract, +1 = they repel.
VectorFieldCenter[0 0 0]Position of the vector-field origin.
VectorFieldSize[10 10 0]Extent of the vector field, [x y z].
VectorFieldStep1Spacing of the vector-field grid.
VectorSegments4Number of segments per field-line vector.
VectorLength1Length of each field-line vector.
OverlayfalseDraw the vector field as an overlay (always on top).
Energydouble (read-only)Potential energy of the charge pairs, J (dependent).

Examples

Two attracting charges

clf; view(3); axis equal; grid on;
a = phx.Body("Position", [-3 0 0]);
b = phx.Body("Position", [ 3 0 0]);

phx.Monopole([a b], "Charge", [1; -1], "Attractivity", -1);

sim = phx.Simulation(gca, "Gravity", [0 0 0]);
sim.step(4, 800, 5);
