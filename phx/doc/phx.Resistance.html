phx.Resistance

Resistance

Class  phx.Resistance  |  Superclass: phx.base.Object

Description

phx.Resistance applies a resistance (drag) force to one or more bodies, depending on the velocity of each body relative to a surrounding environment. The force is a polynomial in the relative speed, F = f(1)·v⁰ + f(2)·v¹ + …, acting opposite to the motion. The environment can itself move and rotate, modelling wind, water current or a rotating medium.

Creation

Syntax

res = phx.Resistance(bodies) res = phx.Resistance(___,Name,Value)

Description

res = phx.Resistance(bodies) creates a resistance element and assigns it to all given bodies. All assigned bodies share the same environment parameters.

Input Arguments

ArgumentTypeDescription
bodiesphx.Body (1×n)Bodies the resistance acts on.

Properties

PropertyType / DefaultDescription
VelocityFactors0Polynomial coefficients of the drag force in relative speed, [f0 f1 f2 ...].
EnvironmentVelocity[0 0 0]Linear velocity of the environment, m/s.
EnvironmentOrigin[0 0 0]Origin of the environment (reference point for the twist).
EnvironmentTwist[0 0 0]Angular velocity of the environment, rad/s.
ForceVectorSize0.001Force-vector multiplication factor for drawing.
OverlayfalseDraw the resistance vectors as an overlay (always on top).

Examples

Quadratic air drag on falling bodies

clf; view(3); axis equal; grid on; camlight headlight;

b1 = phx.Body("Position", [0 0 20], "Shape", {"Box", "Density", 1});   % 1 kg
b2 = phx.Body("Position", [2 0 20], "Shape", {"Box", "Density", 5});   % 5 kg

% Drag proportional to v^2 (f0 = 0, f1 = 0, f2 = 0.5)
phx.Resistance([b1 b2], "VelocityFactors", [0 0 0.5]);

sim = phx.Simulation;
sim.step(3, 300, 1);

Constant wind

phx.Resistance([b1 b2], "VelocityFactors", [0 2], ...
    "EnvironmentVelocity", [5 0 0]);   % 5 m/s wind along x
