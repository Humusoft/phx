phx.assembly.scatter

Bodies scattered randomly inside a box region

Function  phx.assembly.scatter  |  Package: phx.assembly

Description

phx.assembly.scatter creates n dynamic bodies with the given shape at uniformly random positions inside a box region. The region spans −x/2..x/2 and −y/2..y/2 in the ground plane and 0..z upwards (the same convention as the inner space of phx.assembly.arena). With a positive Spacing the positions are sampled by rejection, so the bodies do not overlap at the start. Random numbers are drawn from the global generator (like rand), so a layout is reproduced by seeding it first with rng.

Creation

Syntax

bodies = phx.assembly.scatter(shape,n) bodies = phx.assembly.scatter(shape,n,Name,Value) bodies = phx.assembly.scatter(ax,___)

Description

bodies = phx.assembly.scatter(shape,n) creates the bodies in the current axes and returns them as a 1×n phx.Body array. The shape is either a cell array of a phx.shape.* class name and constructor arguments — a new shape is then created for every body, so shapes with random geometry such as phx.shape.Rock give natural variety — or a phx.shape.* object shared by all bodies.

bodies = phx.assembly.scatter(ax,___) draws into the given axes instead of the current axes; an empty target ([]) creates the bodies without graphics.

Options

OptionDefaultDescription
Region[2 2 1]Dimensions [x y z] of the box that the body centres are drawn from.
Spacing0Minimum distance between body centres (0 = no limit). When the requested count does not fit, the error phx:scatter:regionFull is raised.
RandomOrientationfalseGive every body a uniformly random orientation.
Color—Body colors: one common 1×3 RGB color or an n×3 matrix with one color per body. Applied over the shape color.
Friction[0.5 0 0]Friction coefficients of all bodies.
Position[0 0 0]World position of the region frame.
Orientationeye(3)World rotation of the region as a 3×3 rotation matrix.
EulerAngles[0 0 0]World rotation of the region as Euler angles (z→y→x order), an alternative to Orientation.

Examples

A pile of rocks dropped into an arena

clf; view(3); axis equal; grid on; camlight headlight;

phx.assembly.arena("Size", [8 8 1]);

rng(0);
rocks = phx.assembly.scatter({"Rock", "Radius", 0.4}, 30, ...
    "Region", [7 7 4], "Spacing", 0.8, "Position", [0 0 1]);

sim = phx.Simulation;
sim.step(3, 600, 6);
delete(sim);
