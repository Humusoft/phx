phx.shape.STL

STL imported shape

Class  phx.shape.STL  |  Superclasses: phx.base.Shape, phx.base.ShapeMesh

Description

phx.shape.STL loads its visual appearance from an STL file specified by the Source property. The mesh can be decimated (Details), scaled (Scale) and recentred to its bounding-box origin (Centered). The collision shape is selected by Envelope and can be a bounding box, cylinder, sphere, convex hull or concave triangle mesh ("concavef" uses flipped face winding); mass properties are computed from the actual mesh volume and Density.

Creation

Syntax

shape = phx.shape.STL shape = phx.shape.STL(Name,Value)

Description

shape = phx.shape.STL("Source",file) loads a mesh from an STL file. Set Envelope to choose how the collision shape is approximated and Details below 1 to decimate the mesh.

Properties

Geometry

PropertyType / DefaultDescription
SourcestringSTL source file name. Setting it loads the mesh.
Details1Mesh detail reduction factor (1 = full detail, <1 decimates).
Scale[1 1 1]Scale factor applied to the imported mesh.
CenteredtrueMove the origin to the centre of the bounding box.
Envelope"box"Collision envelope: "box", "cylinder", "sphere", "convex", "concave" or "concavef".
Density1000Volumetric density, kg/m³.

Appearance properties are inherited from phx.base.ShapeMesh and skeletal properties from phx.base.Shape.

Examples

Import a decimated part with a convex hull

phx.extra.Viewer("clear", "DefaultCameraPosition", [10 10 10]);

shape = phx.shape.STL("Source", "bracket.stl", "Details", 0.5, ...
    "Scale", [1 1 1]*0.01, "Envelope", "convex", "Density", 7800);

b = phx.Body("Position", [0 0 3], "Shape", shape);

sim = phx.Simulation;
sim.step(3, 300, 1);
