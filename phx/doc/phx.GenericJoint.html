phx.GenericJoint under development

Generic joint

Class  phx.GenericJoint  |  Superclass: phx.base.Joint

Under development. This class is functional but still evolving; its API and behaviour may change in a future release.

Description

phx.GenericJoint realizes a six-degree-of-freedom constraint with independent linear and angular limits on each axis. With all limits set to zero a given axis is locked; opening a limit frees motion along or about that axis. It generalizes the fixed, prismatic and revolute joints.

Creation

Syntax

joint = phx.GenericJoint(bodyA,bodyB) joint = phx.GenericJoint(___,Name,Value)

Description

joint = phx.GenericJoint(bodyA,bodyB) connects two bodies at their origins with all six degrees of freedom locked. Open them with LinearLimits and AngularLimits.

Properties

PropertyType / DefaultDescription
PointA[0 0 0]Connection point in the local space of body A.
PointB[0 0 0]Connection point in the local space of body B.
LinearLimits[0 0 0; 0 0 0]Lower and upper linear limits per axis (2×3): [min; max] for x y z.
AngularLimits[0 0 0; 0 0 0]Lower and upper angular limits per axis (2×3), rad.

Reaction feedback and the MutualCollisions switch (collisions between the connected bodies, default false) are inherited from phx.base.Joint.

Examples

Allow sliding along the local X axis

clf; view(3); axis equal; grid on; camlight headlight;

a = phx.Body("Type", "static", "Position", [0 0 2]);
b = phx.Body("Position", [0 0 2]);

% Free translation along x in [-1, 1]; everything else locked
phx.GenericJoint(a, b, "LinearLimits", [-1 0 0; 1 0 0]);

sim = phx.Simulation;
b.applyForce([20 0 0]);
sim.step(2, 200, 1);
delete(sim);
