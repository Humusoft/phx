phx.shape.Mesh

Custom shape with texture

Class  phx.shape.Mesh  |  Superclasses: phx.base.Shape, phx.base.ShapeMesh

Description

phx.shape.Mesh wears a user-supplied triangular mesh defined by the Vertices, Normals, Faces and UV properties, optionally textured. The collision shape and mass properties are modeled as an ideal block based on the axis-aligned bounding box of the mesh, scaled by Scale.

Creation

Syntax

shape = phx.shape.Mesh shape = phx.shape.Mesh(Name,Value)

Description

shape = phx.shape.Mesh(Name,Value) creates a shape from the given mesh data. Supply at least Vertices and Faces; provide Normals for shading and UV with a Texture for texturing.

Properties

Geometry

PropertyType / DefaultDescription
VerticesN×3 doubleVertex coordinates.
NormalsN×3 doubleVertex normals.
FacesM×3 doubleTriangle vertex indices.
UVN×2 doubleVertex texture coordinates.
Scale[1 1 1]Scale factor applied to the mesh.
Density1000Volumetric density, kg/m³ (applied to the bounding box).

Appearance properties are inherited from phx.base.ShapeMesh and skeletal properties from phx.base.Shape. Mesh uses the "flat" style by default.

Examples

A body from a custom triangulation

phx.extra.Viewer("clear", "DefaultCameraPosition", [10 10 10]);

[X, Y, Z] = sphere(16);
fv = surf2patch(X, Y, Z, "triangles");

b = phx.Body("Shape", phx.shape.Mesh("Vertices", fv.vertices, "Faces", fv.faces));

sim = phx.Simulation;
sim.step(2, 200, 1);
