phx.shape.Globe

Globe shape

Class  phx.shape.Globe  |  Superclasses: phx.base.Shape, phx.base.ShapeMesh

Description

phx.shape.Globe is a sphere geometry built from a meridian/parallel grid, which carries proper texture coordinates — making it the right choice for an image-mapped sphere (a planet, a ball with a print). The collision shape and mass properties are modeled as an ideal sphere. For an untextured sphere, prefer the lighter phx.shape.Sphere.

Creation

Syntax

shape = phx.shape.Globe shape = phx.shape.Globe(Name,Value)

Description

shape = phx.shape.Globe creates a unit globe with default parameters. phx.shape.Globe(Name,Value) sets properties with name-value arguments.

Properties

Geometry

PropertyType / DefaultDescription
Diameter1Globe diameter, m.
Radiusdouble (dependent)Globe radius, m. Setting it updates Diameter.
Segments32Number of meridian and parallel segments.
Density1000Volumetric density, kg/m³.

Appearance properties are inherited from phx.base.ShapeMesh and skeletal properties from phx.base.Shape.

Examples

A textured planet

b = phx.Body("Shape", {"Globe", "Diameter", 2, "Segments", 48, "Texture", "earth.jpg"});

sim = phx.Simulation(gca, "Gravity", [0 0 0]);
b.AngularVelocity = [0 0 0.5];
sim.step(6, 600, 5);
