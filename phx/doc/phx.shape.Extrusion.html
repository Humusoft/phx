phx.shape.Extrusion

Extrusion shape

Class  phx.shape.Extrusion  |  Superclasses: phx.base.Shape, phx.base.ShapeMesh

Description

phx.shape.Extrusion sweeps a 2-D Profile along a 3-D Spine curve, optionally scaling it per spine point. The collision shape is a convex or concave envelope (Envelope); the mass properties are computed from the actual triangular mesh.

Creation

Syntax

shape = phx.shape.Extrusion shape = phx.shape.Extrusion(Name,Value)

Description

shape = phx.shape.Extrusion(Name,Value) creates an extruded shape. Supply a Spine and a Profile; the Scale may be applied uniformly or per spine point.

Properties

Geometry

PropertyType / DefaultDescription
SpineN×3 doubleSpine curve; each row is a 3-D point along the sweep.
ProfileM×2 doubleCross-section profile; each row is a 2-D point. For a closed profile, repeat the first point at the end.
Scale1×2 | N×2 doubleProfile scale: a single 2-element vector, or one row per spine point.
Axis"z"Modeling axis: "x", "y" or "z".
Envelope"convex"Collision envelope: "convex" or "concave".
Density1000Volumetric density, kg/m³.

Appearance properties are inherited from phx.base.ShapeMesh and skeletal properties from phx.base.Shape.

Examples

Extrude a square profile along a straight spine

phx.extra.Viewer("clear", "DefaultCameraPosition", [10 10 10]);

profile = [-0.5 -0.5; 0.5 -0.5; 0.5 0.5; -0.5 0.5; -0.5 -0.5];
spine   = [0 0 0; 0 0 2];

b = phx.Body("Shape", phx.shape.Extrusion("Spine", spine, "Profile", profile));

sim = phx.Simulation;
sim.step(2, 200, 1);
See also phx.Body, phx.shape.Revolution, phx.base.Shape
