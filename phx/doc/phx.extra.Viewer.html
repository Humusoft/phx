phx.extra.Viewer

Enhanced viewer

Class  phx.extra.Viewer  |  Superclass: handle

Description

phx.extra.Viewer turns a MATLAB axes into an interactive viewer for a PHX scene: mouse and keyboard camera navigation, picking and editing of bodies, a textured sky background, basic preset views, an on-screen text overlay and a built-in free-run mode that steps a phx.Simulation in real time.

The viewer attaches itself to the figure that owns the axes and installs the interaction callbacks. Only one viewer lives on a figure at a time — creating a new one replaces the previous. Deleting the axes (or calling close) tears the viewer down.

Creation

Syntax

viewer = phx.extra.Viewer viewer = phx.extra.Viewer(ax) viewer = phx.extra.Viewer("clear") viewer = phx.extra.Viewer("newfigure") viewer = phx.extra.Viewer(___,Name,Value) [viewer,ax] = phx.extra.Viewer(___)

Description

viewer = phx.extra.Viewer attaches a viewer to the current axes (gca).

viewer = phx.extra.Viewer(ax) attaches the viewer to the given axes or UIAxes ax.

viewer = phx.extra.Viewer("clear") clears the current figure and creates a fresh UIAxes for the viewer; "newfigure" creates it in a new uifigure instead.

[viewer,ax] = phx.extra.Viewer(___) also returns the axes the viewer is attached to — handy with "clear"/"newfigure" so you can draw bodies into ax.

Input Arguments

ArgumentTypeDescription
axaxes | UIAxes | "clear" | "newfigure" Target axes, or a keyword that creates one. Default is gca.

displayText

displayText(viewer,text) displayText(viewer,"") displayText(viewer,text,mode) displayText(viewer,text,mode,maxLines) displayText(___,fontSize,fontColor)

Shows text (a string or a column vector of strings, one per line) as an overlay in the corner of the viewer; "" removes it. mode selects how new text combines with the current content: "replace" (default), "below" (append like a log) or "above". maxLines caps the kept lines in the log modes (default 10). fontSize (default 16) and fontColor (default [1 1 1]) apply when the label is first created.

Examples

Attach a viewer to the current axes

clf; view(3); axis equal;
phx.Body("Type", "static");
phx.Body("Position", [0 0 3]);

viewer = phx.extra.Viewer;             % interactive navigation + sky background

Fresh viewer with a preset camera, then build the scene

[~, ax] = phx.extra.Viewer("clear", ...
    "DefaultCameraPosition", [0 -70 0], "ViewMode", "plain");

box = phx.Body(ax, "Position", [0 0 10], "Shape", {"Box", "Size", [4 4 4]});

sim = phx.Simulation(ax);
sim.step(5, 500, 5);

Show a heads-up text overlay

displayText(viewer, "PHX viewer ready");
displayText(viewer, "t = 0.00 s", "below");   % append a log line
displayText(viewer, "");                       % remove the overlay
Tip. Press F5 in the viewer to free-run the scene without writing a step loop. Tune the cadence with FreeRunStep and FreeRunSpeed (Inf runs as fast as possible).
