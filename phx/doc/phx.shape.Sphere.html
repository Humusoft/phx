phx.shape.Sphere

Sphere shape

Class  phx.shape.Sphere  |  Superclasses: phx.base.Shape, phx.base.ShapeMesh

Description

phx.shape.Sphere is a sphere geometry. The visual appearance is an icosahedron subdivided to an adjustable level of detail; the collision shape and mass properties are modeled as an ideal sphere independent of the visual detail.

Creation

Syntax

shape = phx.shape.Sphere shape = phx.shape.Sphere(Name,Value)

Description

shape = phx.shape.Sphere creates a unit sphere with default parameters. phx.shape.Sphere(Name,Value) sets properties with name-value arguments.

Properties

Geometry

PropertyType / DefaultDescription
Diameter1Sphere diameter, m.
Radiusdouble (dependent)Sphere radius, m. Setting it updates Diameter.
Division3Number of icosahedron subdivisions (level of visual detail).
Density1000Volumetric density, kg/m³.

Appearance properties are inherited from phx.base.ShapeMesh and skeletal properties from phx.base.Shape.

Examples

Simple falling sphere

phx.extra.Viewer("clear", "DefaultCameraPosition", [10 10 10]);

b = phx.Body("Position", [0 0 3], ...
    "Shape", {"Sphere", "Diameter", 0.5, "Division", 4, "Density", 2700});

sim = phx.Simulation;
sim.step(2, 200, 1);
